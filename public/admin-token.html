📚 Google Apps Script での使い方

  1. 上記の「アクセストークン」を 📋 コピー ボタンでコピー
  2. Google スプレッドシート → 拡張機能 → Apps Script を開く
  3. setAccessTokenManually() 関数の中に貼り付け
  4. その関数を実行してトークンを保存
  5. runNow() を実行してデータ同期をテスト
